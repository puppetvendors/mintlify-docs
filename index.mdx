## Overview

The PuppetVendors API allows you to programmatically manage vendor data, update reports, and trigger recalculations for your multi-vendor Shopify store.

**Base URL:** `https://api.puppetvendors.com`

## Rate Limits

The API is rate-limited to **100 requests per minute** per IP address. Exceeding this limit returns HTTP 429.
